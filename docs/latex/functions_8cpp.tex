\hline

\hline
\end{DoxyParams}

\begin{DoxyItemize}
\item \mbox{\hyperlink{classNo}{No}} switch case principal, realizadas 6 tipos de operações diferentes\+:
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item Adicionar uma tarefa
\item Editar uma tarefa existente Neste caso é aberto outro switch case com novas opções\+: A. Editar informações de uma tarefa (Um novo switch case é aberto com os atributos editáveis\+: id, título, descrição, prioridade, data de vencimento e \char`\"{}voltar\char`\"{}) B. Mover tarefa C. Excluir tarefa D. Voltar
\item Ver informações de uma tarefa
\item Ordenar tarefas por prioridade (utiliza os algoritmos de ordenação Bubble Sort e Selection Sort)
\item Exibir o Quadro Kanban
\item Salvar o Quadro Kanban Neste caso é aberto um novo switch case com opções de salvamento\+: A. Arquivo \textquotesingle{}bin\textquotesingle{} existente B. Arquivo \textquotesingle{}bin\textquotesingle{} novo C. Arquivo \textquotesingle{}txt\textquotesingle{} D. Voltar
\item Sair
\item Caso uma opção inválida seja digitada, é solicitado uma nova opção ao usuário até que seja digitada uma das existentes. 
\end{DoxyEnumerate}
