template<typename T>
class ListaDuplamenteEncadeada< T >

Classe que implementa uma lista duplamente encadeada.

A classe ListaDuplamenteEncadeada possui um ponteiro para o primeiro elemento (head) e um ponteiro para o último elemento (tail) da lista. Ela permite adicionar elementos no início (pushFront) e no final (pushBack), remover o primeiro elemento (removeFront) e o último elemento (removeBack), verificar se a lista está vazia (isEmpty), imprimir os elementos da lista (print), obter o tamanho da lista (getSize), remover um valor específico (removeValue) e limpar toda a lista (clear).

Parâmetros do template
TO tipo de dado armazenado na lista.
